19. Time

The time(3m) module provides functions for querying and manipluating time values.

19.1. Time functions

The time_current_millis function
Synopsis 

  #include <mba/time.h>
  uint64_t time_current_millis(void);
Description
The time_current_millis function returns the current time in milliseconds since Jan 1, 1970 as a 64 bit integer. It works equally well on UNIX and Windows.
