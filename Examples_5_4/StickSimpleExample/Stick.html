Not Provided

myspace module

Namespace.

myspace.Add function

Add message-1 and message-2 together.

Lua

string return_value = myspace.Add(string v1, string v2)

C++

std::string ::myspace::Add(const std::string& v1, std::string const& v2)

Parameters

return_value
string type: result
v1
string type: message-1
v2
string type: message-2

myspace.ArrayAdd function

Add v2 to each member of v1.

Lua

array<string> v1 = myspace.ArrayAdd(array<string> v1, string v2)

C++

void ::myspace::ArrayAdd(std::vector<std::wstring>& v1, const std::wstring& v2)

Parameters

v1

v1
array<string> type: string array.
v2
string type: string

myspace.HashAdd function

Lua

C++

void ::myspace::HashAdd(std::map<std::wstring,int>& v1, int v2)

Parameters

v1
